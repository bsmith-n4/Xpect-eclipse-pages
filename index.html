--- title: Xpect ---

Xpect is a unit- and integration-testing framework based on JUnit.

Store test data in text files of any format.

Integrate seamlessly with Eclipse tools, language tests and CI Builds.

Compare Test Results

Xpect uses textual expectations and embeds them into DSL documents, allowing for using the Eclipse comparison editor to inspect failed tests and to fix outdated expectations.



Human Readable and Verifiable

Xpect sits neatly within your workflow by allowing for embedded test expectations whithin code comments. Write tests in a format that's both easy to understand and can be verified automatically by a test suite.
