--- title: Xpect ---

Xpect is a unit- and integration-testing framework based on JUnit.

Store test data in text files of any format.

Integrate language tests with CI Builds.

Goal

Xtext is a framework for the development of programming languages and domain-specific languages.

Xpect is for testing and supporting the process of designing such languages.
